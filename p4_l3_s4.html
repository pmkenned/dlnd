Character-wise RNN Notebook

You can get the notebook with the character-wise RNN from our public GitHub repo in the intro-to-rnns folder.

To clone the entire repository to your machine: 
git clone https://github.com/udacity/deep-learning.git

This code requires TensorFlow 1.0, so make sure you upgrade if you're using an older version. Play with the network, improve it, train it on your own text. This thing is for you to build off of.

If you find ways to improve it, make a pull request and we'll add it in.
